## What is Agent Academy?

Agent Academy is a free, structured curriculum that teaches developers how to work effectively with AI coding agents like OpenCode and Codex. Instead of scattered blog posts and tool docs, you get a clear learning path across nine modules — from understanding what agents are, through prompt engineering, context engineering, and agent skills, all the way to MCP servers, subagent delegation, and security hardening.

## Who is this for?

**New to AI coding agents?** Start at Module 1 to understand what agents are, how they work, and when to use them. By Module 2, you will have a working agent setup ready for real tasks.

**Already using agents casually?** Jump to the module that fills your gaps. Most developers benefit from Modules 3-5 on prompt engineering, context engineering, and agent skills — the techniques that turn inconsistent results into reliable workflows.

## Where to start

:::tip[Recommended starting point]
**Module 1: Introduction** is the best place to begin. It takes about 30 minutes and gives you the mental model you need for everything that follows. From there, Module 2 walks you through setting up your agent environment step by step.
:::

If you already have an agent installed and want to improve your results, skip ahead to **[Module 3: Prompt engineering](/03-prompt-engineering/overview/)** — it covers the techniques that make the biggest difference in day-to-day agent use.
